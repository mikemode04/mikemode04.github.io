Machine Learning for Financial Prediction

Research Question

Can we predict the direction of EUR/USD daily returns with accuracy above 50%? The Meese-Rogoff puzzle (1983) showed economic models fail — but ML classification might work.

Hypothesis: By predicting only direction (up/down) rather than magnitude, we can achieve accuracy significantly above 50% and build a profitable trading strategy.

Methodology

  • Features: Lagged returns (1, 5, 20 days), RSI(14), MACD signal, Bollinger Band %B
  • Models: Logistic Regression, Random Forest, XGBoost, Neural Network (MLP)
  • Validation: Walk-forward analysis — train 250 days, test 20 days, roll forward

# Fair Value Gap Detection def detect_fvg(highs, lows): gaps = [] for i in range(2, len(highs)): if lows[i] > highs[i-2]: # Bullish FVG gaps.append({'idx': i, 'type': 'BULL', 'zone': (highs[i-2], lows[i])}) elif highs[i] < lows[i-2]: # Bearish FVG gaps.append({'idx': i, 'type': 'BEAR', 'zone': (highs[i], lows[i-2])}) return gaps

The Distance Effect

Daily loadings down 2.7 million barrels vs pre-disruption. But West-to-East trade (Atlantic → Asia) surged from 2 to 3+ million bpd. Houston to Singapore = 3x longer than MEG to Singapore. Same barrels, 3x the ships needed.

VLCCs correcting, but clean products (LR2, MR) keep strengthening. Refinery geography doesn't match consumption — refined products must ship from Asian/MEG export refineries to Western demand. Trade lanes structurally longer.
